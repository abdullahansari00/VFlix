1) Run python executable file provided along with project.
	Select the option "Add python to path" and then install

2) Install django by typing following command in command prompt
	"pip install django"

3) Extract the "vflix" folder from given zip file

4) Change your path to working directory i.e. "vflix" folder using "cd" command
	eg: " cd C:\Users\username\env_site\vflix"

5) To run the project type following command in command prompt
	"python manage.py runserver"

6) Open any web browser and type "http://127.0.0.1:8000/" in url section and run the project

The project suggests movies for you on the basis of your watched movies list.
Movies data was taken from imdb website
